# SiamRPN-VGG
Combination of siamrpn and VGG
Download the pretrained weights: [google drive](https://drive.google.com/file/d/1cyxKQY8ELRHTyvMW1SFHVDLrTtla8tLp/view?usp=sharing)

*VOT: VOT testing folder.

*train: training folder.

EAO on VOT2018: 0.21
